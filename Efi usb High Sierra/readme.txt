*****************************************************

	This "EFI" folder is designed for i5-3320m models
	but it can be used also by other x230 models.
	If you can't boot, try without slide=0 boot flag.

*****************************************************

	How to use:
		copy EFI folder into your usb's efi partition
		Boot from usb, use -v flag for debugging
